## 🗺️ 学习路线图

我的学习路径主要遵循由浅入深的原则，大致分为以下几个阶段：

### **阶段一：基础奠基**
*   **数学基础**：线性代数、微积分、概率论与数理统计。
*   **编程语言**：Python及其核心科学生态（NumPy, Pandas, Matplotlib）。
*   **机器学习基础**：了解经典机器学习算法（线性回归、逻辑回归、SVM、决策树等）。

### **阶段二：深度学习入门**
*   **核心概念**：神经网络、前向/反向传播、激活函数、损失函数、优化器。
*   **框架掌握**：熟练使用 **PyTorch** 和/或 **TensorFlow** 进行模型开发。
*   **计算机视觉入门**：CNN（LeNet, AlexNet, VGG, ResNet），图像分类任务。
*   **自然语言处理入门**：RNN, LSTM, GRU，词嵌入，文本分类任务。

### **阶段三：专项深入与拓展**
*   **计算机视觉**：目标检测（YOLO, Faster R-CNN），图像分割（U-Net），生成模型（GAN, Diffusion）。
*   **自然语言处理**：Transformer架构，预训练模型（BERT, GPT），序列到序列模型。
*   **模型优化**：模型剪枝、量化、知识蒸馏等部署优化技术。
*   **其他领域**：强化学习、图神经网络、多模态学习等。

> **注**：你可以在 `Notes/` 或 `Learning-Path/` 目录下找到每个阶段的详细笔记和资源链接。
